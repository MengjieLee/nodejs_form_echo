1、该脚本将浏览器发来的数据汇总并返回一个简单的HTML文档，在文档中以HTML表格的形式将那些数据显示出来。


2、在8080端口监听浏览器的连接请求并且只处理浏览器用HTTP POST 方法发送到/form这个URL的表单数据。


3、该脚本保存在一个名为formecho.js文件中，示例见form.html。
